## 🚀 使用方法

### 一键安装
```bash
curl -fsSL https://raw.githubusercontent.com/xurenlu/aipipe/main/install.sh | bash
```

### 配置通知
编辑 `~/.config/aipipe.json` 文件，配置 AI 服务器和通知方式。

### 启动监控
```bash
# 基本使用
./aipipe -f /var/log/app.log --format java

# 系统服务（Linux）
sudo ./install-systemd.sh
```
